# $BLUEDOG - Blue Dog Democrats Meme Coin Website

A Next.js website for the $BLUEDOG meme coin launching on pump.fun. Blue Dog fights corporate overpricing and advocates for consumer rights.

## Features

- Next.js 14+ with App Router
- TypeScript
- Tailwind CSS for styling
- Framer Motion for animations
- Fully responsive design
- All 13 Blue Dog Rules
- Complete tokenomics information
- Roadmap and community sections

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

## Project Structure

```
/
├── app/              # Next.js app directory
│   ├── about/       # About page
│   ├── rules/       # Blue Dog Rules page
│   ├── tokenomics/  # Tokenomics page
│   ├── roadmap/     # Roadmap page
│   ├── how-to-buy/  # How to buy guide
│   ├── security/    # Security & transparency
│   ├── community/   # Community page
│   ├── faq/         # FAQ page
│   └── whitepaper/  # Whitepaper
├── components/       # React components
├── public/          # Static assets
│   └── images/      # Images (add blue-dog-capitol.jpg here)
└── netlify.toml     # Netlify configuration
```

## Image Setup

Add your Blue Dog Capitol image to `public/images/blue-dog-capitol.jpg`. This image will be used as the hero background.

## Deployment

This project is configured for Netlify deployment. The `netlify.toml` file contains the necessary build configuration.

1. Push code to Git repository
2. Connect repository to Netlify
3. Deploy automatically on push
